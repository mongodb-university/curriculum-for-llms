1. Hello! In this video, you'll learn how to create and insert documents into a collection in MongoDB.

2. There are two methods that we can use to insert documents into a collection: insertOne() and insertMany().

3. Let's start by inserting a single document by using the insertOne() method.

4. To use the insertOne() method, you will append it to the database and collection name, like this: db.collection.insertOne()

5. While connected to our database called training, we'll call the insertOne() method on the grades collection by using db.grades.insertOne().

6. But what if the grades collection doesn't exist in our database yet? Luckily, MongoDB automatically creates the collection for you when you use insertOne(). It's important to remember this when inserting data so that you don't accidentally create unnecessary collections in your database.

7. Now, we pass the document that we want to insert as a parameter inside the insertOne() method. We'll run the command.

8. The result that's returned confirms that the insertOne() statement was successful. Note that the insertedId field tells us the _id value of our newly inserted document.

9. Every document in a collection must have an _id field, and its value must be unique. If you don't provide an _id, like in our example, then MongoDB will automatically generate one for you.

10. What if we want to insert more than one document at once? MongoDB has a method called insertMany(). It looks like this:

11. As its name suggests, insertMany() inserts multiple documents. We pass in the array of documents we want to insert, each separated by a comma.

12. Let's open the shell and examine the insertMany() expression. First, use the insertMany() method on the grades collection with db.grades.insertMany(). Remember, insertMany() expects an array of documents, and each document must be separated by a comma.

13. After we run the command, we'll get a message confirming that the documents have been inserted.

14. Like insertOne(), MongoDB auto-generates a unique _id value for each of the new documents.

15. Let's recap what you learned in this video: First, we inserted a single document by using the insertOne() method. Next, we inserted multiple documents separated by commas by using the insertMany() method. Finally, we learned that if we don't specify an _id value when inserting a document, it will be auto-generated by MongoDB. To practice this, check out our learning activity for this unit.
